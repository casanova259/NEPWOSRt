# Anurag Jha — Portfolio

A modern, minimal, and premium personal portfolio built with the modern JavaScript/TypeScript ecosystem.

## Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [lucide-react](https://lucide.dev/)
- **Language:** TypeScript

## Features

- **Modern UI/UX:** Clean design with glassmorphism elements, micro-interactions, and premium hover states.
- **Dynamic Theming:** Built-in dark/light mode toggle.
- **Smooth Animations:** Scroll reveals, layout animations, and fluid state transitions.
- **Interactive Elements:** Features like the GitHub commit graph visualization and custom easter eggs (Konami code, Neko cat cursor).
- **Fully Responsive:** Optimized for all screen sizes.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Customization

You can easily customize the content of this portfolio by editing `config/site.ts`. The architecture separates configuration from components, allowing you to update your experiences, projects, skills, and links in one central file.
